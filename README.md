💎💎 SIMPLE EXPENSE TRACKER 💎💎

LIVE LINK : https://mnsvi.github.io/Expense-tracker/

A minimalist, single-page web application designed to log and display expenses dynamically.

---

✨ Features

* Expense Input: Quick fields to add an expense name and its corresponding amount.
* Dynamic Display: Appends and displays added expenses instantly on the screen upon clicking "Add Expense".
* Card UI: A clean, centered container design featuring custom rounded corners.

---

🛠️ Tech Stack

* HTML5: Structured layout for inputs and displays.
* CSS3: Custom styling, centering, and container layout.
* JavaScript (ES6): DOM manipulation to dynamically display user inputs.
